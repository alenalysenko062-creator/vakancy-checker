# JobCheck — детектор мошеннических вакансий

ИИ-сервис для проверки вакансий с HH.ru, Авито или из текста.
Анализирует 18 красных флагов + ищет отзывы о работодателе в интернете.

## Быстрый старт (локально)

### 1. Клонировать / распаковать проект

```bash
cd jobchecker
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate       # Mac/Linux
# или
venv\Scripts\activate          # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Добавить API ключ Anthropic

**Mac/Linux:**
```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

**Windows (PowerShell):**
```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

Или создайте файл `.env` и используйте python-dotenv (опционально).

### 5. Запустить сервер

```bash
python main.py
```

Открыть в браузере: **http://localhost:8000**

## Деплой на Timeweb (VPS)

```bash
# На сервере
git clone ... && cd jobchecker
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt

# Добавить ключ в /etc/environment или ~/.bashrc
echo 'export ANTHROPIC_API_KEY=sk-ant-...' >> ~/.bashrc
source ~/.bashrc

# Запустить через systemd или просто:
nohup python main.py &
```

Для production рекомендуется nginx как reverse proxy перед uvicorn.

## Что анализируется

**18 красных флагов в 5 группах:**

1. **Деньги и условия** — сверхзарплата, платный вход, размытая оплата, % без оклада
2. **Компания и контакты** — анонимность, нет адреса, личные контакты, подозрительный домен
3. **Текст вакансии** — шаблонность, срочность, завышенные обещания, размытые обязанности
4. **Схемы** — МЛМ, финансовые схемы, удалёнка без опыта 100к+, дропшиппинг
5. **Процесс найма** — «берём всех», без трудового договора, оформление «потом»

**Веб-ресёрч работодателя:**
- Поиск отзывов на DreamJob, Otzovik
- Поиск «[компания] мошенники / развод»
- Проверка на Rusprofile

| Источник | Метод |
